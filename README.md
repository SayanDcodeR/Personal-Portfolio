# Sayan Singha - Personal Portfolio

A modern, responsive, and highly interactive personal portfolio website built from scratch to showcase my skills, projects, and professional journey as a Full-Stack Developer. 

## 🚀 Features

- **Custom Canvas Animation**: A high-performance, mathematics-based particle network animation running in the background, utilizing HTML5 `<canvas>`.
- **Dark & Light Mode**: Seamless theme switching with persistent dynamic styling across all UI components and animations.
- **Responsive Layout**: Pixel-perfect design that adapts flawlessly to mobile, tablet, and desktop screens using CSS Grid and Flexbox.
- **Glassmorphism UI**: Premium card designs with rounded corners, subtle inset borders, and smooth hover effects.
- **Functional Contact Form**: Integrated with Web3Forms to allow direct email messaging without a backend server.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure and layout.
- **CSS3**: Custom variables, responsive grids, and advanced styling (No external CSS frameworks used).
- **Vanilla JavaScript**: DOM manipulation, theme toggling, and the custom physics engine for the background animation.

## 📂 Sections Included

1. **Hero**: Quick introduction, profile picture, and call-to-action buttons.
2. **About Me**: Academic background and hackathon achievements.
3. **Skills**: Categorized technical skills (Languages, Web Technologies, Databases, Tools/DevOps).
4. **Projects**: Detailed showcase of highlighted projects (JanMitra, TastyBite) with source code links.
5. **Certifications & Extracurriculars**: Proof of continuous learning and community involvement.
6. **Contact**: Direct email form and social links.

## 💻 How to Run Locally

Because this project is built with pure web technologies, no build tools or servers are required to view it locally!

1. Clone this repository:
   ```bash
   git clone https://github.com/SayanDcodeR/Personal-Portfolio.git
   ```
2. Navigate into the project folder.
3. Double-click the `index.html` file to open it in any modern web browser.
